# Telegram Gifts Dune Dashboard — Chinese Market Edition

> 🇨🇳 English summary for TON Society Footstep #1227

## Overview

This dashboard provides a **Chinese-community-focused** lens on Telegram Collectible Gifts on-chain activity. While #1226 covers the global audience, this version adds China-specific analytical layers:

1. **CNY denomination** — All charts include CNY-equivalent views using P2P exchange rates
2. **Chinese whale tracking** — Community-labeled whale wallet identification
3. **Localized heatmap** — Trading activity shown in Beijing time (UTC+8)
4. **Chinese-language documentation** — Full tutorial + FAQ for Chinese-speaking newcomers

## Key Charts

| # | Chart | China-specific value |
|---|-------|---------------------|
| 1 | Daily volume overview | CNY toggle for Chinese traders |
| 2 | Marketplace share | Highlights Getgems/Disintar (popular with CN users) |
| 3 | Whale leaderboard | Can be annotated with community-known CN wallets |
